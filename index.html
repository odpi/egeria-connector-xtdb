    Crux Connector

    Pluggable repository for Egeria, with native support for historical metadata.

    Get started Go to GitHub

    Persistent

    Uses Crux's pluggable architecture to support a variety of underlying storage back-ends including S3, RocksDB, Apache Kafka, LMDB, JDBC and more.

    Historical

    Leverages Crux's temporal graph queries to provide native support for storing historical information and answering temporal queries.

    Performant

    Currently the highest-performance open source persistent repository for Egeria across all operations: read, write, update, search and purge.

    Highly Available

    Currently the only open source Egeria repository to be capable of running as a highly-available service.
